## Resumo

- O vídeo apresenta o **Claude Code**, uma ferramenta de IA da Anthropic voltada para programação, que agora pode ser utilizada de forma **gratuita e ilimitada** através da integração com a ferramenta de código aberto **Ollama** (0:00-0:40). O autor demonstra como instalar e configurar essa dupla para automatizar tarefas complexas de desenvolvimento diretamente no terminal ou em IDEs (1:19-2:56).
- O vídeo explica que é possível utilizar agentes de IA de forma totalmente local e offline utilizando a ferramenta **Ollama**. Segundo o vídeo, ao baixar e rodar modelos localmente, como o **Qwen 3.5**, o usuário elimina a necessidade de depender de modelos na nuvem que possuem limites diários de uso (23:51-24:45).

## Casos de Uso Práticos

- **Agentes Personalizados:** Criação de um agente especializado em análise de dados que converte arquivos _Excel/CSV_ em painéis visuais interativos em _HTML_ (4:13-10:24).
- **Webs Interativas:** Uso do _Claude Code_ em conjunto com a ferramenta _Flow_ (_Nano Banana 2_) para criar sites profissionais com animações complexas, como uma página de produtos do McDonald's (11:55-16:57).
- **Habilidades (Skills) Avançadas:** Instalação de plugins para dar ao agente capacidade de gerenciar arquivos do sistema, demonstrado pela criação de um aplicativo local de limpeza de disco (_Clean Navi_) (16:57-23:51).
- **Uso Ilimitado em Local:** Como configurar e utilizar modelos de IA (como o _Qwen 3.5_) rodando **localmente** na máquina do usuário para evitar limites de uso de APIs em nuvem (23:51-28:50).

## Passo a passo

O workflow (fluxo de trabalho) demonstrado no vídeo para utilizar o Claude Code gratuitamente e sem limites consiste em:

1. **Instalação do Ollama (0:17-0:40):** O usuário instala o _Ollama_ em sua máquina para permitir a execução de modelos de IA locais.
2. **Instalação/Configuração do Claude Code (0:42-1:07):** Através do terminal, o usuário instala o _Claude Code_ e o vincula ao _Ollama_ usando um comando específico.
3. **Seleção do Modelo (1:07-1:24):** O usuário escolhe um modelo de IA (na nuvem ou local) para atuar como o 'cérebro' do agente de programação.
4. **Definição do Escopo (1:24-1:40):** O usuário navega para a pasta do projeto no terminal para que o agente tenha acesso apenas aos arquivos necessários.
5. **Execução e Automatização (1:40-3:50):** O agente _Claude Code_ é iniciado e o usuário fornece instruções em linguagem natural para que ele analise dados, crie código, gerencie arquivos ou construa aplicações inteiras (como demonstrado nos casos de uso do vídeo).

## Instalando skills no agente

Para instalar novas **habilidades (skills)** no agente _Claude Code_ e expandir suas capacidades, você deve seguir estes passos diretamente no seu terminal:

1. **Abrir o Menu de Habilidades:** Digite `slash` (`/`) no terminal para abrir o menu de opções e selecione **`plugin`** ou **`skills`**.
2. **Adicionar do Marketplace:** Escolha a opção de instalar a partir do **`marketplace`** ou **`add from repository`**.
3. **Inserir o Repositório:** Cole a URL do repositório que contém as habilidades desejadas (como o repositório oficial de habilidades do _Claude_).
4. **Confirmar a Instalação:** O agente irá processar e instalar as habilidades no seu projeto atual. Você poderá vê-las organizadas em uma pasta específica do projeto (19:54-21:04).
