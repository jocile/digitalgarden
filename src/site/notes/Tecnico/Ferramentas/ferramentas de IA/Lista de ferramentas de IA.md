#Inteligencia-artificial 
#Ferramentas

| Ferramenta                                                | Descrição                                                                                                                                                                                           |
| :-------------------------------------------------------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [[Tecnico/Ferramentas/ferramentas de IA/Antigravity/Antigravity\|Antigravity]]                                           | Um **IDE Agêntico** desenvolvido pelo Google, onde agentes de IA atuam de forma proativa no código, terminal e navegador, enquanto o desenvolvedor atua como supervisor.                            |
| [Context7](https://context7.com/)                         | é um servidor que utiliza o **Model Context Protocol (MCP)** para fornecer documentação de código e exemplos atualizados e específicos de versões para modelos de linguagem (LLMs) e editores de IA |
| [[Tecnico/Ferramentas/ferramentas de IA/Crewia\|Crewia]]                                                | Framework de orquestração multiagente que permite criar equipes de IAs com papéis específicos (como pesquisador ou escritor) para resolver tarefas complexas colaborativamente.                     |
| [Mastra](https://mastra.ai/docs)                          | Framework de código aberto projetado para construir agentes de IA e fluxos de trabalho (workflows), especialmente focado em ecossistemas TypeScript e Next.js.                                      |
| [[Tecnico/Ferramentas/ferramentas de IA/Notebooklm\|Notebooklm]]                                            | Ferramenta do Google que organiza e analisa informações de fontes específicas (PDFs, sites, vídeos), permitindo criar resumos e tirar dúvidas focadas apenas nesse conteúdo.                        |
| [[Tecnico/Ferramentas/ferramentas de IA/Ollama\|Ollama]]                                                | Plataforma que facilita a execução de modelos de linguagem (LLMs) como Llama 3 e Mistral **localmente** no seu computador, garantindo privacidade e eliminando custos de API.                       |
| [[Tecnico/Ferramentas/ferramentas de IA/OpenClaw\|OpenClaw]]                                              | Assistente pessoal agêntico de código aberto capaz de gerenciar e-mails, controlar o navegador e executar comandos no sistema para automatizar tarefas do dia a dia.                                |
| [[Tecnico/Ferramentas/ferramentas de IA/Perplexity\|Perplexity]]                                            | Mecanismo de busca baseado em IA que processa linguagem natural para fornecer respostas diretas e contextuais, sempre citando as fontes de onde as informações foram extraídas.                     |
| [TRAE](https://www.trae.ai/)                              | Um IDE adaptativo (ambiente de desenvolvimento) impulsionado por IA, desenvolvido pela ByteDance, focado em aumentar a produtividade e a compreensão do contexto do código.                         |
| [Tavily](https://www.tavily.com/)                         | Motor de busca otimizado especificamente para agentes de IA e LLMs, permitindo que sistemas autônomos realizem pesquisas na web de forma rápida e estruturada.                                      |
| [Superwhisper](https://superwhisper.com/)                 | é uma ferramenta de ditado baseada em Inteligência Artificial que converte voz em texto com foco em **privacidade e processamento local**                                                           |
| [Stitch - Design with AI](https://stitch.withgoogle.com/) | permite criar designs de interface de usuário (UI) para aplicativos móveis e web a partir de comandos de texto ou imagens de referência                                                             |
