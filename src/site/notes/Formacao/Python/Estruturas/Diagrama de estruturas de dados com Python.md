# Diagrama de estruturas de dados com Python

![](Diagrama%20de%20estruturas%20de%20dados%20com%20Python.svg)

Explicações detalhadas:

- **Listas**:
  - `Ordenadas e Mutáveis`: Os elementos podem ser alterados e mantêm a ordem de inserção.
  - `Permite Duplicatas`: Elementos podem ser repetidos na lista.
  - `Acessadas por Índices`: Os elementos são acessados através de índices numéricos.

- **Tuplas**:
  - `Imutáveis`: Não podem ser alteradas após a criação.
  - `Acessadas por Índices`: Os elementos são acessados da mesma forma que em listas.
  - `Usadas para Dados Constantes`: São úteis para dados que não devem ser modificados.

- **Sets**:
  - `Coleção Não Ordenada`: Os elementos não possuem uma ordem específica.
  - `Conjunto de Elementos Únicos`: Cada elemento é único no conjunto.
  - `Operações de Conjuntos`: Suporta operações como união, interseção e diferença de conjuntos.

- **Dicionários**:
  - `Chave-Valor`: Cada elemento é uma combinação de chave e valor.
  - `Mutáveis`: Os elementos podem ser alterados após a criação.
  - `Permite Chaves Únicas`: Cada chave em um dicionário é única.

Este mapa mental expandido fornece uma visão mais detalhada e organizada das características e usos de cada tipo de estrutura de dados em Python. Ele pode ser útil para estudantes, desenvolvedores e entusiastas de Python que desejam entender melhor como cada estrutura de dados funciona e em que situações elas são mais adequadas.
