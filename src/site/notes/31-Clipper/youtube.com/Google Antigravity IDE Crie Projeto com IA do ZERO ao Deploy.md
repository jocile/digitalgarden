## Resumo

Neste tutorial completo (0:00), **Matheus Battisti** ensina a criar um projeto web do zero até o deploy utilizando a nova **IDE do Google, o Antigravity**, focando na abordagem de _vibe coding_ com IA.

### **Destaques do Vídeo:**

- **Introdução ao Antigravity:** (0:00 - 2:09) O vídeo apresenta o projeto final: um **gerador de desculpas** inteligente, desenvolvido em menos de 15 minutos com _Next.js_ e _Tailwind CSS_.
- **Configuração e Vibe Coding:** (2:10 - 5:00) Passos para baixar e instalar o IDE, conectar com a conta Google para usar o Gemini e instalar a extensão do navegador.
- **Criação do Projeto:** (5:01 - 20:00) Matheus utiliza prompts de IA para gerar a estrutura inicial, telas e navegação, enfrentando e corrigindo erros de compatibilidade (Linux/Windows).
- **Funcionalidades Avançadas:** (20:01 - 33:04) Implementação de **autenticação**, integração com a **API da OpenAI** (geração de desculpas reais), banco de dados e gerenciamento de estado das desculpas (criar, excluir, copiar).
- **Deploy Profissional:** (33:05 - 51:53) Passo a passo para colocar o projeto no ar em uma **VPS** (_Hostinger_), configurando domínio, SSL e DNS, além de utilizar o _PM2_ para gerenciar a aplicação.
